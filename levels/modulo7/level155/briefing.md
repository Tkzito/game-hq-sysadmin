# Nível 155 — Persistência de Dados (Volumes)

## 🎮 Contexto do Freela
Banco de dados MySQL rodando em container perde todas as informações gravadas toda vez que o container é atualizado ou reiniciado.

## 🎯 Comando-Chave
``docker run -d -v /var/data/mysql:/var/lib/mysql --name db-mysql mysql``
