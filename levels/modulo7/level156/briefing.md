# Nível 156 — Injeção de Variáveis de Ambiente

## 🎮 Contexto do Freela
A API de pagamentos se conecta a servidores de teste ou de produção dinamicamente dependendo da variável `NODE_ENV`.

## 🎯 Comando-Chave
``docker run -d -e NODE_ENV=production -e DB_PASS=secret_key api-server``
