# Nível 228 — Redis Sentinel para Failover Automático

## 🎮 Contexto do Freela
Configurar Sentinels para monitorar e promover a réplica a primária automaticamente caso o nó master Redis caia.

## 🛠️ Missão
Executar ou configurar o cenário conforme os tópicos de infraestrutura.

## 🎯 Comando-Chave
`sentinel.conf, sentinel monitor mymaster, sentinel failover`

## 🎯 Critério de Sucesso
* Executar as tarefas descritas na missão com sucesso de acordo com as diretrizes do ecossistema.
