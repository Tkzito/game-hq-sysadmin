# Nível 61 — Leitura Linha por Linha

## 🎮 Contexto do Freela
Ao lidar com automação, frequentemente precisamos ler arquivos de texto estruturados linha por linha para processá-los de maneira robusta.

## 🛠️ Missão
Escreva o script `/home/operator/ler_linhas.sh` que lê o arquivo `/home/operator/nomes.txt` linha por linha usando o loop `while read -r` e imprime cada linha no formato `Nome: <conteudo_da_linha>`.

## 📝 Comandos Úteis
*   `while read -r linha; do ... done < arquivo.txt`

## 🎯 Critério de Sucesso
O script `/home/operator/ler_linhas.sh` deve ler o arquivo linha por linha e imprimir as saídas formatadas.
