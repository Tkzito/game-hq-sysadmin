# Nível 114 — Verificando Consumo de Memória: O que a RAM Esconde

## 🎮 Contexto do Freela
Após encerrar o processo problemático, o servidor ainda responde lento. Suspeita-se que a memória RAM está esgotada e o sistema passou a usar swap intensamente, o que degrada brutalmente a performance.

## 🛠️ Missão
Analise o consumo de memória física e swap usando o comando `free -h` no terminal.
O sistema validará se você executou a verificação de memória.

## 🎯 Comando-Chave
`free -h`, `free -m`

## 🎯 Critério de Sucesso
* Executar o comando `free -h` (ou similar) no terminal para inspecionar o uso de memória.
