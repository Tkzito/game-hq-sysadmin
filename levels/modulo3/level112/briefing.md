# Nível 112 — Monitoramento Interativo com htop

## 🎮 Contexto do Freela
O `ps aux` dá uma fotografia do momento, mas o servidor da OralTech apresenta picos intermitentes. Precisamos de uma visão em tempo real que atualize continuamente.

## 🛠️ Missão
Instale o utilitário `htop` usando o gerenciador de pacotes do sistema (`apt-get` / `apt`).
Abra o `htop` no terminal para monitorar os processos ativos em tempo real. O sistema validará se a ferramenta foi instalada e executada.

## 🎯 Comando-Chave
`htop`, `sudo apt-get update && sudo apt-get install -y htop`

## 🎯 Critério de Sucesso
* Instalar o `htop`.
* Executar o `htop` no terminal.
