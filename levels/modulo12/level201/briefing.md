# Nível 201 — Virtual Private Cloud (VPC)

## 🎮 Contexto do Freela
Criar uma rede isolada virtual privada com blocos CIDR 10.0.0.0/16 e associar a tabela de rotas e Internet Gateway.

## 🛠️ Missão
Executar ou configurar o cenário conforme os tópicos de infraestrutura.

## 🎯 Comando-Chave
`aws ec2 create-vpc, internet-gateway, route-table`

## 🎯 Critério de Sucesso
* Executar as tarefas descritas na missão com sucesso de acordo com as diretrizes do ecossistema.
