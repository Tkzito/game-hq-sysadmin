# Nível 172 — Proxy Reverso para APIs

## 🎮 Contexto do Freela
Configurar o Nginx para encaminhar requisições do endpoint /api para a porta 8080 onde a API de pagamentos está rodando.

## 🛠️ Missão
Executar ou configurar o cenário conforme os tópicos de infraestrutura.

## 🎯 Comando-Chave
`proxy_pass, location /api/ {}`

## 🎯 Critério de Sucesso
* Executar as tarefas descritas na missão com sucesso de acordo com as diretrizes do ecossistema.
